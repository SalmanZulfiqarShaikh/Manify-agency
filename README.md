# Manify – Next‑Gen Cinematic Agency Landing

Website: https://www.manify.vercel.app

Manify is a next‑gen single‑page agency website with a clean, minimal UI, smooth modern animations, and a cinematic dark experience. It showcases web development, AI automation (n8n workflows, agents), SEO, and app development on one polished page.

## Tech Stack

- React 
- Tailwind CSS  
- Framer Motion 
- Swiper JS
